# Volunteer Registration System

This is a complete MERN stack Volunteer Registration System featuring a modern Light Mode Tailwind CSS design.

## Features
- **Frontend**: React (Vite), React Router, Tailwind CSS, Redux Toolkit, React Hook Form
- **Backend**: Node.js, Express, MongoDB Atlas, Mongoose, JWT Authentication
- **User Roles**: Admin and Volunteer

## How to Run the App

1. **Start the Backend**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Start the Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Admin Credentials

To access the Admin Dashboard, use the following seeded credentials to log in via the `/login` route on the frontend:

## Volunteer Credentials

You can test the volunteer side by clicking "Register" on the frontend and filling out the volunteer registration form. After successful registration, you will be redirected to the Volunteer Dashboard.
